第三章 C编程基础知识

常量和变量

常量：编译时已经确定，在程序运行时不发生变化 。常数不是常量；常量除了常数（整型常量、实型常量），还包括字符常量、字符串常量、符号常量等。
变量：内存中的一段存储区域，程序运行过程中其值可变；变量必须先定义才能使用。
存储地址的变量称为指针变量。

小端模式：存数据时先存低字节，再存高字节。

字节数：char 1，short 2，double 8，其余常用变量都为4；TC中int为2。

变量定义的位置应该在同级别的执行语句之前。（C99取消了这条规定）

标识符：
（1）英文字母、数字、下划线组成，不能以数字开头；
（2）区分大小写；
（3）不允许用关键字作为标识符；
关键字指已经赋予一定含义的字符序列，如int、float
（4）标识符有长度限制，超过限制，后面的字符不起作用；
（5）见名知意；

每个变量都有值和地址两个属性；但不是仅有这两个属性，还可以有类型等属性
变量的值指的是变量在内存中所存的内容；
变量的地址指的是变量在内存中所处的位置，其起始位置称为变量的地址。

定义变量的目的是为了存储数据。

赋值：在给变量分配空间的任务完成后，再给变量存放数据，赋值就是存储：把赋值号右边表达式的值计算出来然后存储到左边变量的内存空间。（两次执行）
赋初值：再给变量分配空间的时候就向其中存放数据。（一次执行完）

基本数据类型

C语言的数据类型分为基本类型和构造类型，基本类型指的是系统固有的类型，也是常用的类型；构造类型指的是用户自定义出来的类型。

整型数据

程序中用到整型常量时，可以用3种进制：十进制、八进制和十六进制。
100的表示方式：十进制100，八进制0144、十六进制0x64；注意程序中不允许使用二进制

%d表示十进制输出整数，%o表示八进制，%x或%X表示十六进制，加上L或l表示输出长整数，加上h表示输出短整数，无符号整数用%u。

实型数据

实型常量

带小数点的常量称为实型常量。

（1）小数形式 3.14 当一个数是纯小数时，小数点前面的0可以省略。
（2）指数形式 1.25E-2 即1.25*10^-2 E前数字，E后整

实数只能用十进制表示。

实型常量有单精度F/f（float）和双精度L/l（double）两种方式，有效数字分别为7位、15位，最后一位近似位。

实型数据的存储

实型数据的存储，对于单精度和双精度的数据，都是以浮点数的方式存储。
格式为：(符号)X M X2^n，其中，n是指数，M必须满足：1.0 <= M < 2.0
30.0表示为 +1.875 X 2^4

float型数据存储空间的分配：

符号位(0或1) 指数部分（n+127） 小数部分（M-1）
占1位[第31位] 占8位[第30位~第23位] 占23位[第22位~第00位]
部位存储待补充P43

实数的存储不都是精确的。

%f小数形式输出，默认输出6位小数（其他形式%.2f,指定两位小数）
%e(%E)指数形式输出，小数点前有且仅有1位非0的有效数字。

字符型数据

单个的字符是字符常量

字符放 ‘ ‘ 中，单引号有新的含义-定界符；

反斜线 \ 转义字符。

C语言认为字符也有大小，即其ASCII码。

48 - 0
57 - 9
65 - A
90 - Z
97 - a
122 -z
大小写字母差了32

字符串

置于双引号中，双引号中可以没有字符，表示空串。

C语言只有字符串常量，没有字符串变量。

存储字符串时，最后多存一个空字符’\0’。
空字符的ASCII是0，表示’\0’。

空字符和空格不是同一个字符，空为0，空格为32。
“A”和’A’不同，字符串结尾多一个空字符。

输出：%s

对于数据的存取：char型数据存储其ASCII码值，short、int、long型数据存储补码，float和double型数据按IEEE754标准所规定的格式存储。

符号常量和常变量

符号常量

符号常量的定义是一条命令，不是语句，后面不需要分号。

编译预处理命令，称为宏定义。

使用方便、便于修改、增加可读性。

符号常量只是一个符号，不是变量，内存中没有空间，所以不能赋值。
#define PI = 3.1415926 X
#define PI 3.1415926 √

符号常量名通常使用大写字母

常变量

const开头

常变量定义时必须初始化，为变量赋初值。

常变量的定义是一条语句，后面有分号。

常变量只可以赋初值，不能赋值，因为赋值等于改写。

定义常变量之后，程序中可以随时使用它，但不能改变它。

运算符和表达式

算术运算符

求余运算的结果，其符号应该与 % 前面那个数的符号相同。

求余的两个量必须是整数。

+（正号）、—（负号）是单目运算符，只需要一个运算量。（即操作数）

左结合性：当两个运算符优先级别相同时，先算左面。

两个整数运算，最终结果还是整数，一般向零取整而非四舍五入。
若想结果是实数，则要将一个数变成实数。5./9

赋值运算符和赋值表达式

左值： 可以出现在赋值号左面的式子。

左值必须有内存空间且允许赋值，常用的左值是变量，但常变量不是左值
常变量允许赋初值，但不允许赋值

赋值表达式也有值，一个赋值表达式的值，等于赋值后左边变量的值。

其他运算符

自增自减运算符：++、–
表达式++i的值是加1之后的i（先+1），而表达式i++的值是加1之前的i（运算后i+1）。

相当于赋值运算，因此只能作用于变量，故无法作用于表达式和常量

逗号运算符：，
双目运算符，优先级最低，逗号表达式的值等于逗号后面式子的值。先求解前面表达式，后求解后面表达式，值为后面的表达式，可不断嵌套式子1,式子2,式子3······

类型转换运算符：把表达式的值强制转化为指定的类型。
格式：（类型名）（表达式）

数据的类型转换

（1）参与运算的操作数若有char、short型，无条件转为int，若有float，无条件转为double（提高精确度），不管类型是否相同。
（2）若参与运算的操作数类型仍不同，则低类型数据转换为高数据类型。由低到高 int 、unsigned、long 、double

习题

选择

int x = 10,y = 3;printf(“%d,%d\n”,x–,–y);的输出结果（D）
(A)10 2
(B)10,3
(C)9,3
(D)10,2

设有定义“char c = “CHINA”;”,则下面说法正确的是（D）
(A）c是一个字符串变量
(B) c占用6个字节的内存
(C) c的有效字符个数是5
(D) 语句错误，无法编译

属于合法的字符常量的是（B）
(A)’\084’八进制
(B）’\x43’十六进制
(C)’ab’
(D)”\0”

填空

表达式（a = 1, 2, 3, a）是一个（赋值）表达式，其值为（1）

表达式（（a = 3*5，a * 4）,a + 15）的值为（30），执行后a的值为（15）
