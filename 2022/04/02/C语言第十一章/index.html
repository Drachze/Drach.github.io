第十一章 数组

一维数组

定义

定义格式 ： 类型名 数组名 [常量表达式] ;

（1）数组也是一种类型，称为数组类型。
（2）数组名由程序员命名，须遵守标识符的命名规则；
（3）数组名后面[ ]中是一个常量表达式，其中不允许含有变量，但可以使用符号常量。常量表达式的值代表数字组含有的个数，类型应为整型或字符型。
（4）数组元素的下标从0开始计数。
（5）每个元素都是数组定义所规定类型的一个变量。

下标变量

存储

一维数组在内存中占据一段连续的存储空间。

数组名代表一个数组类型的对象，在以下三种使用场合，他都代表整个数组：
（1）测试整个数组的空间大小：sizeof（a）。
&emsp; sizeof在计算地址a + i 会自动对 i 乘以sizeof（int），然后再与 a 相加。
（2）获取整个数组的指针：**&a。
（3）用字符串对字符数组初始化**。
其他场合退化为一个指向数组首元素的指针。a + 1 ，此时a表示一个指向变量a[0]的指针常量。

数组名代表一个对象，即整个数组，不是常量。只有被用作指针时，才是一个指向数组首元素的指针常量。

表达式 a + i 的值（大小）就是a[i]的地址，a的类型决定着表达式的类型。因此，表达式a + i是一个指针，指向a[i],所以*可以用（a + i ）表示 a[i]**。

表示方法

下标法： a[0] &ensp; 编译器仍当作指针法处理
指针法：* a

为赋初值的元素自动置为0。

二维数组

定义

类型名 数组名[常量表达式1][常量表达式2]；

二维数组的元素是一维元素，他的元素的元素是下标变量。

二维数组在内存中占据一段连续的存储空间。

存储较一维数组多出混合法 *（a[1]+2）

初始化

（1）给所有下标变量赋初值 int a[2][3] = {1,2,3,4,5,6} 所有数据写在一对大括号内
（2）每行数据再加一个大括号（多维） int a[2][3] = {{1,2,3},{4,5,6}}
（3）可以省略第一个[]中的值 int a[][3] = {{1,2},{3},{4}} //相当于int a[3][3]
（4）只对部分下标变量赋值。 int a[][3] ={{1,2},{1}} //下标变量的值依次是1、2、0、1、0、0

字符数组

初始化

（1）单字符形式与其他类型数组的初始化相同
（2）字符串形式
&emsp;char a[10] = “string”;占用前面7个内存空间，包括一个空字符'\0'，剩下自动置零
&emsp;char b[] = “string”;只分配七个字节空间（按需分配）
&emsp;char c[3][8] = {“abc”,”123”,”string”}；共分配24个空间，每个一维数组八个字节
